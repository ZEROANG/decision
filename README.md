> **理性的逻辑计算，感性的命运倾向。**

这是一个为“纠结症患者”设计的轻量化决策辅助工具。它不仅能进行严谨的加权评分，还能模拟命运的随机偏转，并配套了智能的动态存档系统。

---

## 🚀 三大决策模式

### 1. ⚖️ 天平决策 (Balance Mode)
* **设计理念**：模拟命运的摆动，而非机械的分值累加。
* **核心逻辑**：基于 `Math.random()` 的随机偏移算法。
* **交互反馈**：系统会根据随机生成的“偏转百分比”生成感性话术（如：“天平瞬间倒向了收益” 或 “天平纹丝不动”）。
* **适用场景**：当你心中已有倾向，只需一个“仪式感”来帮你拍砖。

### 2. 📊 对比决策 (Advanced Mode)
* **设计理念**：多维度加权评分，追求绝对理性。
* **核心逻辑**：依据 `(权重 % * 评分)` 汇总得分。
* **智能建议**：
    * **阶梯话术**：根据分差幅度给出不同强度的建议（如：绝对优势、险胜、不分伯仲）。
    * **可视化排行**：直观展示得分前三名的梯度，辅助深度对比。

### 3. 🎰 随机决策 (Random Mode)
* **设计理念**：绝对公平的概率选择。
* **视觉效果**：内置 3 秒“老虎机”式滚动动画，增加决策的期待感与仪式感。
* **适用场景**：吃什么、去哪玩、谁去拿外卖等日常琐事。

---

## 💾 智能存档系统 (Auto-Save Logic)

为了平衡“快速测试”与“认真记录”的需求，系统设计了一套**柔性存档机制**：

* **⚡ 匿名决策**：无需标题即可开始决策。系统会温馨提示“当前未存档”，不阻断用户使用心流。
* **📝 即时补全**：结果产出后，用户只需在标题框补全名称，系统将**自动触发**静默存档。
* **🛡️ 智能重命名**：
    * **冲突检查**：当标题与已有历史重名时，系统会自动分配 `(1)`, `(2)` 序列号。
    * **打字保护**：打字过程中静默保存，仅在失去焦点（Blur）时执行序号修正，确保输入流畅不跳光标。
    * **另存为机制**：修改历史记录的标题并保存，将自动识别为“新决策”并创建副本。

---

## 🛠️ 技术实现

-   **数据存储**：采用 `localStorage` 进行本地持久化，数据不上传服务器，保护隐私。
-   **动画引擎**：基于 CSS3 `cubic-bezier` 贝塞尔曲线实现的平滑滚动效果。
-   **字符保护**：全局集成 `checkLen` 逻辑，标题严格限制在 20 字符以内，确保 UI 规整。

---

## 📝 使用指南

1.  **输入选项**：根据模式填写维度评分、权重或备选项。
2.  **启动决策**：观察天平摆动、得分对比或转盘滚动。
3.  **命名记录**：若结果有参考价值，填入标题，系统将自动记录至底部的“历史档案”。
4.  **复盘回溯**：点击历史记录可一键找回当时的决策现场。
